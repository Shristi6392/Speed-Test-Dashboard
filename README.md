🚀 Speed-Test-Dashboard – Internet Speed Test & Network Analyzer

This is a modern, interactive web application that allows users to test and analyze their internet performance in real-time. Built with a clean aqua-themed UI, it provides detailed insights into download speed, upload speed, and network latency along with historical tracking and visualization.

🌐 Live Demo/Deployed
🔗 Try the app here:
https://speed-test-dashboard-76ksa9zrxphhgfldvtzhtc.streamlit.app/

📌 Key Features
⚡ Real-Time Speed Testing
Measure Download Speed (Mbps)
Measure Upload Speed (Mbps)
Check Ping / Latency (ms)

📊 Interactive Dashboard
Beautiful speedometer gauge visualization
Clean dark-themed UI with aqua highlights
Instant display of results in metric cards

🧠 Smart Network Analysis
Automatically evaluates connection quality:
Excellent
Good
Average
Slow
Helps users understand performance without technical knowledge

📜 Test History Tracking
Stores previous test results
Displays in a structured table
Enables performance comparison over time

📈 Data Visualization
Line charts for:
Download speed trends
Upload speed trends
Helps identify network fluctuations

🌐 Network Information
Displays public IP address
Basic network diagnostics

⚙️ Custom Settings
Toggle:
Speed meter (ON/OFF)
Smart analysis (ON/OFF)

🛠️ Tech Stack
Technology	
Python-- Core programming language

Streamlit--	Web app framework

Plotly-- Interactive visualizations

Pandas-- Data handling & history tracking

Requests-- API calls (IP fetching)

Speedtest-cli-- Internet speed measurement

🚀 Getting Started (Run Locally)

🔹 1. Clone the repository
git clone https://github.com/Shristi6392/Speed-Test-Dashboard.git
           cd your-repo

🔹 2. Install dependencies
           pip install -r requirements.txt

🔹 3. Run the application
           streamlit run app.py
